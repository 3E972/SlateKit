Slatekit

Open Source Tablet UX Building Kit

Download this project as a .zip file Download this project as a tar.gz file

What is SlateKit?

These are a set of Qt-based UI components, runs on top of Android system, aim for people who crazy enough to create their own tablet UX and leverage existing products.

Get It

You can download the latest release or see the wiki for more detailed instructions.

Build It

You'll need Android NDK as toolchain, and we suggest to base your work on Mer Project or Linaro Ubuntu.

Use It

Check out the sample init script for more details.

Get Involved

Please check out the code and find out how else you can contribute.

Get In Touch

Want to see SlateKit running on your tablet? Having ideas for a custom UI? contact penkia@gmail.com and we'll help you sort it out.
